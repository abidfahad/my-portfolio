## Step 1: Setup and First Commit
- Created local repository
- Initialized Git using `git init`
- Added README.md
- Made first commit

## Step 2: Connect to GitHub
- Created GitHub repository
- Connected local repository using remote origin
- Pushed project to GitHub

## Step 3: Branching Basics
- Created `feature-about-me` branch
- Added `about.md`
- Merged branch into `main`
- Deleted merged branch

## Step 4: Multiple Branches and Pull Requests
- Created `feature-contact`
- Added `contact.md`
- Created Pull Request
- Simulated code review
- Merged PR into `main`

## Step 5: Conflict Resolution Practice
- Created merge conflict in `README.md`
- Resolved conflict manually
- Committed resolved version
- Merged changes successfully

## Step 6: Essential Git Commands
- Used `git checkout --` to discard changes
- Viewed commit history using:
  - `git log --oneline --graph`
  - `git show`
- Practiced Git stash commands:
  - `git stash`
  - `git stash pop`

## Step 7: GitHub Collaboration Features
- Created GitHub Issue
- Linked Pull Request to Issue
- Added labels and assignee
- Closed Issue after merge
